# Brand logos

Drop the supplied logo PNGs here with these exact names:

- `crash-logo.png`  — the white-background CRASH logo
- `jmk-logo.png`    — the JMK Solicitors logo

brand.js renders these automatically (on a white pill, so a white-background
logo reads on the dark top bar) and falls back to the text wordmark if a file
is missing. Tall full-lockup logos are fine — they're scaled to ~30px high in
the top bar.
